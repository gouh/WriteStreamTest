* Existe un archivo llamado "run.sh" que utiliza una imagen de nodejs para correr el script lo que debemos hacer es lo siguiente
* Primero dar permiso al archivo "run.sh"
```bash
chmod +x ./run.sh
```

* Luego ejecutar dicho archivo
